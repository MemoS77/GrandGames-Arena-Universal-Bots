# GrandGames Arena Universal Bots

Приложение позволяющее подключать произвольные движки В качестве ботов к сервису GrandGames Arena: https://arena.grandgames.net

## Как это работает

Используется официальный SDK сервиса https://github.com/MemoS77/GrandGames-Arena-Bots-SDK

Запускается в отдельном процессе AI движок настольной игры, позволяющий делать лучшие ходы по переданной позиции.
При наступлении позиции, требующей хода бота, позиция подготавливается и передается движку. В свою очередь ответ движка конвертируется в формат, понятный сервису.

## Для кого?

Это ПО может использовать и дорабатывать любой разработчик, желающий запустить свой AI движок в качестве бота на сервисе GrandGames Arena для любых не деструктивных целей: развлечение, обучение, проверка уровня AI.
Приветствуется добавление адаптеров для новых движков, будем рады вашим Pull Request.

## Как пользоваться

- Необходимо иметь предуставление о том, как работает node.js вообще и npm в частности

- Cтавим node.js 24 LTS или выше: https://nodejs.org/en
  Устанавливаем зависимости: `npm i`

- Регаем бота (новый аккаунт обязательно с окончанием Bot на GrandGames Arena, иначе токен не будет виден в кабинете и бот будет забанен): https://arena.grandgames.net и получаем токен в прфоиле.

- Прописываем в conf.example.json токен, секции для поддержываемых вашим ботом игр и ссылку на выполняемый файл бота.

- Запускаем. Команды в package.json, измените их под себя.

## Реализованые протоколы

На текущий момент реализованы следующие протоколы:

### UCI

https://en.wikipedia.org/wiki/Universal_Chess_Interface

Это самый популярный интерфейс для шахматных программ.
Примеры реализаций: Stockfish: https://stockfishchess.org/download/

### Gomocup

Протокол для программ в турнирах по гомоку на сервисе https://gomocup.org/
Примеры программ: https://gomocup.org/download-for-developers/#source
